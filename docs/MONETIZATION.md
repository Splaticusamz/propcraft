# Monetization Strategy — PropCraft

## Pricing Tiers

### Free ($0/mo)
- 2 proposals per month
- Basic templates (generic, web dev)
- Markdown export only
- PropCraft watermark on exports

### Pro ($9/mo)
- 10 proposals per month
- All templates (design, marketing, consulting, dev, etc.)
- PDF + DOCX export
- No watermark
- Edit history

### Unlimited ($19/mo)
- Unlimited proposals
- Custom branding (logo, colors, footer)
- Full template library
- Priority generation (faster model)
- API access (coming soon)

## Growth Channels

1. **SEO:** Blog content targeting "freelance proposal template", "SOW generator", "how to write a project proposal"
2. **Reddit:** r/freelance, r/webdev, r/graphic_design — genuine value posts
3. **Indie Hackers:** Build in public, share revenue milestones
4. **Twitter/X:** Freelancer community engagement
5. **Referral:** Give 1 free doc, get 1 free doc

## Revenue Targets

- Week 1: 50 signups, 5 paid ($45-95 MRR)
- Month 1: 500 signups, 50 paid ($450-950 MRR)
- Month 2: $1K+ MRR
